# seiseki

## How to use

1. Copy `.env.default` to `.env` file.

1. Write `UEC_ID` and `UEC_PASSWORD` for .env file.

1. Install dependencies.
    ```
    $ yarn install
    ```
    or
    ```
    $ npm install
    ```

1. Run this program.
    ```
    $ yarn start
    ```
    or
    ```
    $ npm start
    ```

1. Output image file as `seiseki.png`.
